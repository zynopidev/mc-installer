
# ⛏️MC-INSTALLER by Zynopi (beta 1.0)       

This script is used to easily install Minecraft Paper servers.



## Documentation

 - Run this script as administrator.
 - All minecraft servers are started in SCREEN session, so if you decide to start your server again, use command **sudo bash run.sh** to start a minecraft server. This **startup-script** is located in server directory.
 - Do not edit file **run.sh** to ensure properly working.
 - New features and optimalizations will be add soon.
